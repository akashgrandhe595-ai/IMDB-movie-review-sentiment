# IMDB-movie-review-sentiment
IMDB Movie Review Sentiment Analysis using NLP
Overview:
This project performs sentiment analysis on IMDB movie reviews using Natural Language Processing (NLP) and Machine Learning techniques. The model classifies reviews as positive or negative.
Objective:
To build a machine learning model that can accurately predict the sentiment of movie reviews.
Dataset:
- Dataset: IMDB Movie Reviews Dataset
- Contains:
  - Review text
  - Sentiment (Positive / Negative)
Tools & Technologies:
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK / spaCy
- Matplotlib / Seaborn
Steps Involved:
1. Data Loading
2. Text Cleaning (removing punctuation, stopwords)
3. Text Preprocessing (tokenization, stemming)
4. Feature Extraction (TF-IDF)
5. Model Training (Naive Bayes / Logistic Regression)
6. Model Evaluation
Model Used:
- Naive Bayes / Logistic Regression
Results:
- Successfully classified movie reviews into positive and negative categories
- Achieved good accuracy on test data
Example:
Input:
"This movie was amazing and very interesting"
Output:
Positive
Future Improvements:
- Use deep learning models (LSTM, BERT)
- Improve preprocessing techniques
- Deploy as a web application
